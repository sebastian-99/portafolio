Descripción del proyecto

El turimo es algo importante para los ingresos dentro de distintas áreas municipales y el municipio de Lerma, Estado de México, no queria quedarse atras. Una página Web donde se muestra las atracciones y localizaciones locales para dar a conocer que se puede hacer, que se puede ver e inclusive un poco de historia de este município.
